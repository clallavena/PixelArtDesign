Application Pixel's Art Design:
L'application permettra aux utilisateurs de colorier, pixel par pixel, une image qui une fois toute coloriée, se découvrira.

-L'application permettra de prendre une photo via la caméra du téléphone et ainsi l'image prise pourra être coloriable.
